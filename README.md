# go-vocabtxt

Package **vocabtxt** provides tools for working with `vocab.txt` files used with WordPiece tokenizing, for the Go programming language.

## Documention

Online documentation, which includes examples, can be found at: http://godoc.org/github.com/reiver/go-vocabtxt

[![GoDoc](https://godoc.org/github.com/reiver/go-vocabtxt?status.svg)](https://godoc.org/github.com/reiver/go-vocabtxt)

## Import

To import package **vocabtxt** use `import` code like the follownig:

```
import "github.com/reiver/go-vocabtxt"
```

## Installation

To install package **vocabtxt** do the following:

```
GOPROXY=direct go get github.com/reiver/go-vocabtxt
```
